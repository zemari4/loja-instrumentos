Como importar

  1. Exporte a planilha atual clicando em Exportar XLSX na lista de estoque
  2. Edite os produtos na planilha (adicione linhas para criar, edite para atualizar)
  3. Para criar novos produtos, mantenha a coluna id vazia (mas ela deve estar presente)
  4. As colunas categoria e marca devem ter o nome exato cadastrado
  5. Suba o arquivo aqui
